# api_ventaslocal
App donde se implementan el uso y comunicación de microservicios.

## Características

- Funciones de CRUD basicas para Pacientes, Turnos.
- Funcionalidades especificas detalladas en el codigo.
- Arquitectura de Microservicios.
- Uso de Docker.
